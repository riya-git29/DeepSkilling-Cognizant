# REST Employee Management

## Objective

Develop RESTful Web Services using Spring Boot to return Employee and Department data stored in Spring XML configuration.

## Technologies

- Java
- Spring Boot
- Spring Core
- Spring MVC
- REST API
- Maven
- Postman

## REST Endpoints

### Get Employees

GET /employees

Returns all employees.

### Get Departments

GET /departments

Returns all departments.

## Testing

Use Postman to test:

http://localhost:8080/employees

http://localhost:8080/departments

## Output

- Employee list returned as JSON.
- Department list returned as JSON.
- Console displays:
  Department REST Service Called
