# JUnit Basic Testing Exercises

## Objective

This project demonstrates the basics of JUnit 4 testing.

## Exercises Covered

### Exercise 1
- JUnit setup using Maven.

### Exercise 2
- Writing unit tests for Calculator class.
- Testing add, subtract, multiply and divide methods.

### Exercise 3
- Using JUnit Assertions:
  - assertEquals()
  - assertTrue()
  - assertFalse()
  - assertNull()
  - assertNotNull()
  - assertSame()
  - assertNotSame()
  - assertArrayEquals()

### Exercise 4
- Arrange-Act-Assert (AAA) Pattern
- @Before setup method
- @After teardown method

## Technologies

- Java
- Maven
- JUnit 4.13.2

## How to Run

1. Import the project into IntelliJ IDEA or Eclipse.
2. Ensure Maven dependencies are downloaded.
3. Right-click the test classes and select **Run**.
4. Alternatively, execute:

```bash
mvn test
```

## Expected Result

```
Tests run: 8
Failures: 0
Errors: 0
BUILD SUCCESS
```
